◆ keyval_resolve()

keyval* keyval_resolve ( keyval *  kv,
const char *  path,
const char **  rem_path 
)

Resolves a path within a keyval tree to a node. If the node is not found, the rem_path output parameter will contain the suffix of the path that could not be resolved and the last found parent will be returned. Example: keyval_resolve( foo, "a.b.nonexist.c.d.e", &rem_path ) returns the node a.b rem_path is set to nonexist.c.d.e
