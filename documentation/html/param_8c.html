Creates a parameter list with one double parameter and returns. If this is succesful, the status output will be set to "OK", otherwise, the status output will be set to contain a friendly error message. This function can be used as a convenient chainreturn on functions.

Parameters
statusFriendly status message output
nparamParameter count output
paramParameter list pointer output
valueThe value to copy into the parameter list.
Returns
0 when successful. or -1 if not.
